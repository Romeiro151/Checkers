# Checkers
This was my first programming project, developed in Java, consisting of a console-based implementation of the classic Checkers (Draughts) game. The game logic was built using matrices (2D arrays) to represent the board state, manage piece positions, and validate moves.

Through this project, I learned the fundamentals of programming, including control structures, object-oriented principles, and array manipulation. It was an important milestone in developing my problem-solving skills and understanding core Java concepts.
